How to Use The ADAM Project
===========================

Installation
------------

To use **ADAM**, clone this github repository then install it in development mode using pip:

.. code-block:: console
   
   (.venv) $ git clone https://github.com/Naww137/ADAM TARGETDIR
   (.venv) $ pip install -e .
